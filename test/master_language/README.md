# Master Language Tests

This directory contains comprehensive tests for the Master Language module.

## Test Files

- `test_master_ast.py` - Tests for AST classes and functionality
- `test_master_parser.py` - Tests for parser functionality  
- `test_master_interpreter.py` - Tests for interpreter functionality

## Running Tests

Run all master language tests:
```bash
pytest test/master_language/
```

Run specific test files:
```bash
pytest test/master_language/test_master_ast.py
pytest test/master_language/test_master_parser.py
```
